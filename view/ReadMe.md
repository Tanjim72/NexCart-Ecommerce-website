# NexCart E-commerce System

## Product & Inventory Management Module

### Features:
1. **Product Catalog** - Organized by category, brand, and price
2. **Search & Filter** - Smart search with price filters
3. **Product Details** - Basic product information display
4. **Stock Management** - Track and update inventory
5. **Admin Panel** - Add/Edit products easily

### Setup:
1. Place all files in a web server directory
2. Access index.php in browser
3. No database required - uses localStorage

### Usage:
- Browse products in Catalog
- Use Search for specific products
- View details by clicking product
- Manage stock in Stock Management
- Add products in Admin Panel

### Tech:
- PHP for server-side
- JavaScript for client-side
- localStorage for data
- Simple CSS design
